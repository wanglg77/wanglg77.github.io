手腕一动就钻心疼？不是娇气，是你的肌腱在“闹罢工”！这篇“救手指南”请查收

核心提示：你以为只有干重活才会手疼？错！玩手机、抱孩子、敲键盘……这些“不起眼”的小动作，正在悄悄废掉你的手。

楔子：一只被“吓坏”的保温杯

昨天门诊来了位很时髦的年轻宝妈，叫小雅。

她进诊室的时候，左手托着右手，右手悬在半空，一脸的惊恐，眼泪都在眼眶里打转。

“医生，快帮我看看！我的手是不是废了？还是骨折了？或者……是不是长了什么不好的东西？”

我问她怎么了。

小雅说：“早上起来，我刚想去拿床头的保温杯喝口水，手刚握住杯子，还没提起来，手腕那里就像被电击了一样，‘滋’的一下钻心地疼！杯子‘哐当’一声掉地上摔瘪了，我的心也跟着碎了……”

我看着她那只并没有红肿、外表光鲜亮丽的手，淡定地笑了笑：“来，把大拇指包在手心里，握个拳，然后手腕往小拇指那边偏一下。”

小雅照做了，刚偏了一点点，立马惨叫一声：“哎呦！就是这儿疼！疼死我了！”

“行了，破案了。”我一边写病历一边说，“不是骨折，也不是绝症，你这是典型的**‘妈妈手’，学名桡骨茎突狭窄性腱鞘炎**。”

小雅一脸懵：“腱鞘炎？那不是老年人才得的病吗？我才28岁啊！”

各位，千万别觉得腱鞘炎离你很远。在这个**“机不离手”的时代，腱鞘炎早就不是什么老年病、职业病，它已经成了“全民流行病”**。

一、 什么是腱鞘炎？——一场“宝剑”与“剑鞘”的爱恨情仇

要说清楚这个病，咱们得先上一堂极简解剖课。

咱们的手指之所以能灵活地弯曲、伸直，靠的是肌腱（就是咱们俗称的“筋”）牵拉。而为了防止这根“筋”乱跑、乱磨，老天爷给它配了一个保护套，这就叫腱鞘。

你可以把肌腱想象成一把宝剑，把腱鞘想象成剑鞘。

  • 正常情况下：宝剑（肌腱）在剑鞘（腱鞘）里进进出出，润滑油充足，丝般顺滑，你动起来行云流水。

  • 腱鞘炎发生时：你频繁地拔剑、收剑（疯狂刷手机、不停敲键盘、抱娃姿势不对），宝剑和剑鞘之间过度摩擦，摩擦生热，虽然没起火，但是发炎了！

发炎之后，**剑鞘（腱鞘）**会充血、水肿、变厚，通道变窄；**宝剑（肌腱）**也会变粗。

结果就是：粗宝剑卡在窄剑鞘里。

这时候你再动，那就不是丝般顺滑了，那是生拉硬拽。轻则疼痛难忍，重则直接卡住不动（这就叫“扳机指”，手指头弯下去弹不回来，还得用另一只手给它掰直）。

二、 谁是“幕后黑手”？

你以为腱鞘炎是针对你？不，它是针对所有**“不知节制”**的人。

  1. 新手爸妈（尤其是妈妈）：

    • 有个词叫“妈妈手”，专门形容这种情况。因为长时间单手抱娃、喂奶，手腕一直保持一个僵硬的姿势，大拇指过度用力。这就像是一直拉着满弓不放箭，弦能不崩吗？

  2. 手机重度依赖者：

    • 说的就是正在看文章的你！现在手机屏幕越来越大，你单手操作时，大拇指得拼命去够屏幕边缘。这绝对是违反人体工程学的“自残”行为。

  3. 键盘侠与鼠标手：

    • 办公室白领，手指在键盘上跳舞，手腕在桌子上摩擦。一天成千上万次的敲击，肌腱早就累吐血了。

  4. 家务主力军：

    • 拧毛巾、切菜、拖地，这些需要手腕频繁旋转、用力的动作，都是腱鞘炎的温床。

三、 灵魂拷问：我到底是不是腱鞘炎？

来，咱们做一个Finkelstein试验（握拳尺偏试验），这是骨科医生的“照妖镜”，在家就能做：

  1. 伸出你的手，掌心向内。

  2. 把大拇指弯曲，放在掌心。

  3. 其余四个手指握拳，把大拇指紧紧包在里面。

  4. 保持这个姿势，手腕向小拇指的方向（即向下）压。

判决时刻：

如果你的手腕在大拇指那一侧，产生剧烈的疼痛（注意，是剧痛，不是那种牵拉感），那恭喜你（或者是同情你），大概率中招了。

四、 既然得了，怎么治？

很多人一听是炎症，就问：“医生，我要不要吃抗生素？”

停！此炎非彼炎！

这是无菌性炎症，不是细菌感染，吃头孢、阿莫西林一点用都没有！

治疗腱鞘炎，医生的法宝通常有三级：

第一级：佛系治疗——“懒”字诀

这是最有效，但也最难做到的。

休息，休息，还是TMD休息！

既然是累出来的病，歇着就是最好的药。少抱娃（让老公抱），少玩手机（用支架），少做家务（买洗碗机）。

如果你实在控制不住自己，医生会给你开个支具，强行把你的手腕和大拇指固定住。就像给不听话的孩子穿上束缚衣，强迫它休息。

第二级：药物攻势——外敷内服

  • 外用：扶他林（双氯芬酸二乙胺乳胶剂）、氟比洛芬凝胶贴膏等。哪里痛贴哪里，让药物透皮吸收去消炎。

  • 内服：如果疼得睡不着，可以吃点非甾体抗炎药（比如布洛芬、塞来昔布）。记住，是为了消炎镇痛，不是为了让你吃了药继续玩手机！

第三级：精准打击——传说中的“封闭针”

很多患者一听“封闭”就色变：“医生，那是激素啊！会有副作用！会骨质疏松！会变胖！”

辟谣时间：

腱鞘炎打的封闭针，确实含有少量激素（糖皮质激素）和麻药。但是！这是局部注射，药量极小，直接作用在发炎的腱鞘上。它不会让你变成大胖子，也不会让你骨头酥得像饼干。

对于顽固性疼痛，打一针封闭往往有“起死回生”的奇效。

但是注意：一般一年内注射不超过3次，间隔要长。

第四级：终极手段——手术

如果上面都试了，手还是动不了，或者手指彻底卡死，那就得动刀了。

做一个小手术，把那个狭窄的腱鞘切开或者是松解一下。就像把紧绷的皮带松两个扣，宝剑自然就又能拔出来了。

五、 医生教你两招，把腱鞘炎扼杀在摇篮里

既然是“磨”出来的病，咱们平时就要多保养。

动作一：橡皮筋抗阻训练

拿一根橡皮筋（别太粗，把你绑头发的拿下来就行），套在五根手指外缘。然后手指用力张开，撑开橡皮筋，坚持3-5秒，再慢慢合拢。

这能锻炼伸指肌群的力量，平衡肌力。

每天3组，每组10-15次。

动作二：手腕拉伸

手臂伸直，手掌向外（像推墙一样），用另一只手把手指向后掰，拉伸屈肌；然后手掌向内（像招财猫一样），向后掰，拉伸伸肌。

工作一小时，做一次，每次坚持15-30秒。

六、 医生的心里话

在门诊看了这么多腱鞘炎，我发现这不仅仅是一个病，更像是一个生活隐喻。

腱鞘和肌腱的关系，就像我们和生活的关系。

我们总是想要抓得更多——抓工作、抓业绩、抓孩子的教育、抓手机里的信息流。

我们拼命地“拔剑”，却忘了给“剑鞘”喘息的机会。

身体的疼痛，其实是一封求救信。

它在告诉你：慢一点，松一点。

不要等到手握不住杯子的时候，才想起要对自己温柔。

把手机放下，去握一握爱人的手；

把家务分担出去，去摸一摸公园的花草。

毕竟，我们这双手，是用来拥抱幸福的，不是用来为了生活而磨损废掉的。

写在最后

如果看完这篇文章，你下意识地转动了一下手腕，觉得有点酸爽，那么请立刻做两件事：

  1. 放下手机，甩甩手。

  2. 动动你那灵活的（或者不太灵活的）手指，给我点个“在看”和“赞”。

我是你们的医生朋友，关注我的公众号，这里不仅治病，更治愈生活。咱们下期见！
